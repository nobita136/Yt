API Documentation

All endpoints return JSON. Use these in your bots, scripts, or apps.

Base URL https://your-app.onrender.com

Replace with your Render deployment URL. Locally it's http://localhost:5000

GET /search?q={query} Search YouTube videos

Parameters

NameTypeRequiredDescription
qstringRequiredSearch query
limitintegerOptionalNumber of results (default: 12)

Example Request

GET /search?q=lofi+music&limit=5

Example Response

{ "results": [ { "id": "dQw4w9WgXcQ", "title": "Never Gonna Give You Up", "thumbnail": "https://i.ytimg.com/vi/...", "duration": "3:33", "channel": "Rick Astley", "views": 1400000000, "url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ" } ] }
GET /download/audio/{youtube_url} Get best audio + all audio formats

Parameters

NameTypeRequiredDescription
youtube_urlstringRequiredFull YouTube URL (youtu.be or youtube.com)

Example Request

GET /download/audio/https://youtu.be/HNMP1UdjOsE

Example Response

{ "status": "ok", "title": "Video Title", "thumbnail": "https://...", "duration": "3:45", "channel": "Channel Name", "best_audio": { "format_id": "251", "ext": "webm", "acodec": "opus", "abr": 96, "filesize_human": "5.2 MB", "url": "https://googlevideo.com/..." }, "all_audio_formats": [ /* array of audio format objects */ ] }
GET /download/video/{youtube_url} Get all video + audio formats

Parameters

NameTypeRequiredDescription
youtube_urlstringRequiredFull YouTube URL

Response Format Structure

{ "status": "ok", "title": "Video Title", "duration": "1:04", "channel": "Netflix", "thumbnail": "https://...", "best_audio": { /* best audio format object */ }, "formats": { "combined": [ /* video+audio in one stream (360p/480p) */ ], "video_only": [ { "format_id": "137", "resolution": "1920x1080", "ext": "mp4", "height": 1080, "fps": 24, "url": "https://...", // video stream "audio_url": "https://...", // best audio stream "audio_ext": "webm", "has_video": true, "has_audio": false } ], "audio_only": [ /* audio-only streams */ ] }, "formats_flat": [ /* all formats in one flat array */ ], "formats_count": 24, "note": "video_only formats include audio_url for merging" }

⚠️ Important: YouTube serves high-quality video (720p+) as video-only streams. To get video with sound, use url + audio_url and merge with ffmpeg:
ffmpeg -i video.mp4 -i audio.webm -c copy output.mp4
