API Documentation

Three clean endpoints. All responses are JSON. Use them in bots, scripts, or apps.

Base URL https://your-app.onrender.com
GET /search Search

Query Parameters

ParamTypeRequiredDescription
qstringYesSearch keyword or phrase
limitintegerNoNumber of results (default: 12, max: 20)

Example Response

{ "results": [ { "id": "dQw4w9WgXcQ", "title": "Never Gonna Give You Up", "thumbnail": "https://i.ytimg.com/vi/...", "duration": "3:33", "channel": "Rick Astley", "views": 1400000000, "url": "https://youtube.com/watch?v=dQw4w9WgXcQ" } ] }
GET /download/audio?url={youtube_url} Audio

Query Parameter

ParamTypeRequiredDescription
urlstringYesFull YouTube URL (youtu.be or youtube.com/watch)

Example Request

GET /download/audio?url=https://youtu.be/HNMP1UdjOsE

Example Response

{ "status": "ok", "title": "Video Title", "thumbnail": "https://...", "duration": "3:45", "channel": "Channel Name", "best_audio": { "format_id": "251", "ext": "webm", "acodec": "opus", "abr": 96, "filesize_human": "5.2 MB", "url": "https://..." }, "all_audio_formats": [ { "format_id": "140", "ext": "m4a", "acodec": "mp4a.40.2", "abr": 128, "url": "..." }, { "format_id": "251", "ext": "webm", "acodec": "opus", "abr": 96, "url": "..." } ] }
GET /download/video?url={youtube_url} Video

Path & Query Parameters

ParamTypeRequiredDescription
urlstringYesFull YouTube URL (youtu.be or youtube.com/watch)
heightinteger / "best"NoWhen provided, downloads a merged Video+Audio MP4 at that quality (e.g. 1080, 720, 480, 360)

Without &height — Returns JSON with all formats

GET /download/video?url=https://youtu.be/HNMP1UdjOsE { "status": "ok", "title": "Video Title", "duration": "1:04", "quality_options": [ { "height": 1080, "fps": 24 }, { "height": 720, "fps": 24 }, { "height": 480, "fps": 24 } ], "formats": { "combined": [ /* streams with video + audio together */ ], "video_only": [ /* high-quality video streams */ ], "audio_only": [ /* audio streams */ ] }, "formats_flat": [ /* all formats in one array */ ], "formats_count": 23 }

With &height=720 — Downloads merged MP4 file

GET /download/video?url=https://youtu.be/HNMP1UdjOsE&height=720 → Response: binary video/mp4 file (Content-Disposition: attachment) → Video and audio are merged server-side before download
Bot usage:
wget "BASE_URL/download/video?url=YOUTUBE_URL&height=1080" -O output.mp4

Supported height values: 2160 · 1440 · 1080 · 720 · 480 · 360 · 240 · best
