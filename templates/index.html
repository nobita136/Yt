API Documentation

All endpoints return JSON. Base URL: https://your-app.onrender.com

GET /search?q={query}&limit={n} Search YouTube

Parameters

NameTypeRequiredDescription
qstringRequiredSearch query
limitintegerOptionalResults count (default: 12)

Response

{ "results": [{ "id": "dQw4w9WgXcQ", "title": "...", "thumbnail": "...", "duration": "3:33", "channel": "...", "views": 1400000000, "url": "https://youtube.com/watch?v=..." }] }

GET /download/merge/{youtube_url}?height={p} ⭐ Download merged video+audio MP4

Parameters

NameTypeRequiredDescription
youtube_urlstringRequiredFull YouTube URL in the path
heightinteger or "best"OptionalMax video height: 1080, 720, 480, 360 etc. Default: best

Examples

GET /download/merge/https://youtu.be/HNMP1UdjOsE?height=1080 GET /download/merge/https://youtu.be/HNMP1UdjOsE?height=720 GET /download/merge/https://youtu.be/HNMP1UdjOsE
✅ This endpoint returns a binary MP4 file (Content-Disposition: attachment).
yt-dlp downloads the best video + best audio stream and merges them server-side with ffmpeg.
In your bot: wget "BASE_URL/download/merge/YOUTUBE_URL?height=720" -O video.mp4
